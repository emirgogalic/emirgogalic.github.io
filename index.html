What is mining?

Mining is the process of adding transaction records to cryptocurrencies public ledger of past transactions. It is the process that keeps the cryptocurrency network alive.

Mining is intentionally designed to be resource-intensive and difficult so that the number of blocks found each day by miners remains steady. This so called “proof-of-work” concept is what gives legitimacy to the ledger contained in the network. This is why it’s done with either specialized hardware or GPUs - miners.

Where is the catch?

Mining is also used to introduce currency into the system: Miners are paid any transaction fees as well as a "subsidy" of newly created coins. This serves as motivating people to provide security for the system.

How it works?

Submitted transactions

Transactions are submitted into the network by users sending tokens/coins

Blocks get formed

Miners like us form packages of the transactions into minable blocks

Mining the blocks

Blocks are mined by randomly guessing secret which is hard to find but easy to check

Miners are awarded

First to find this secret is awarded block reward and block is appended to the ledger held by the network, since other nodes can easily check if the solution is correct
